# 🎓 LearnoHub

### Turning Curiosity into Structured Learning with AI.


LearnoHub is an AI-powered educational platform that transforms static learning resources into personalized learning experiences. The platform enables students and lifelong learners to upload educational content, generate structured learning modules, track progress, discover relevant opportunities, and explore new domains through AI-generated learning paths.

Our mission is to make learning more accessible, engaging, and personalized by combining intelligent content organization, AI-assisted learning, and progress tracking into a single platform.

---

## 🚀 Features

### 📚 AI-Powered Learning Paths

* Enter any topic or domain of interest.

* Explore content topic-wise and module-wise.

## 🎯 Problem Statement

Traditional learning resources are often scattered across multiple platforms and presented in unstructured formats, making it difficult for learners to stay organized and focused.

LearnoHub addresses this challenge by transforming educational content into structured, personalized learning journeys that help learners discover, understand, and master new skills efficiently.

---

## 💡 Solution

LearnoHub serves as a centralized learning ecosystem where users can:

* Upload and organize learning materials.
* Generate AI-powered learning modules.
* Explore new domains through structured roadmaps.
* Track learning progress.
* Discover educational opportunities.
* Access curated learning resources and videos.

## 🛠️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/LearnoHub.git
cd LearnoHub
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
node server.js
```

### Open in Browser

```text
http://localhost:3000
```
